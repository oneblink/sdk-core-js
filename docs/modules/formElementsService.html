findFormElement

  • findFormElement(elements: FormElement[], predicate: (element: FormElement, elements: FormElement[]) => boolean, parentElements?: FormElement[]): FormElement | void
    • Defined in src/formElementsService.ts:88

    Iterate over all form elements and return an element that matches a predicate, also iterating over nested form element (e.g. page elements). Will return undefined if no matching element is found.

    Example

     
    const formElement = formElementsService.findFormElement(
      form.elements,
      (formElement) => {
        return formElement.id === '123-abc'
      },
    )

    Parameters

      The form elements to iterate over

          Returns boolean

    Returns FormElement | void
